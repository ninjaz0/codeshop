# 🚀 Virtual Goods Store (Codeshop)

> **Note: This project was fully constructed and developed with the assistance of AI.**

This is a lightweight, modern virtual goods automated delivery and redemption platform. Using a decoupled frontend-backend architecture, the system focuses on the automated delivery and transaction workflow of virtual items (such as software activation keys, membership vouchers, tutorial texts, etc.). Users do not need to pay with real fiat currency; instead, they can increase their account balance by redeeming "Redemption Codes" generated by the admin in the backend, and use that balance to purchase digital goods in the store.

---

## ✨ Core Features

- **🌍 Perfect Multilingual Support (i18n)**
  - The system is built-in and fully adapted to 5 languages: Simplified Chinese, Traditional Chinese, English, Japanese, and Russian.
  - Users can freely switch languages on the homepage at any time.

- **💳 Closed-Loop Virtual Economy**
  - **Recharge System**: Admins generate "Redemption Codes" with specific face values in the backend.
  - **Balance Consumption**: Users input the redemption code to get account balance, and use it to complete the shopping loop in the store.

- **📦 Smart Automated Delivery System**
  - **"Unique Key" Mode**: One key per machine. For each sale, the system automatically deducts stock and issues a unique key to the user.
  - **"Fixed Content" Mode**: Infinite stock. All users purchasing this product will receive the same download link or fixed text.

- **🔐 Strict Invitation and Security Mechanism**
  - **Invitation Code Registration**: New user registrations require an "Invitation Code" generated by the admin backend, effectively controlling abuse.
  - **Transactional Data Protection**: Purchase deductions, stock deductions, and redemption code usages are fully protected by SQLite database transactions, strictly preventing concurrent overselling and data anomalies.

- **🛠️ Powerful Admin Dashboard**
  - Product listing/delisting management, and virtual key stock importing.
  - Batch generation and one-click copying of recharge redemption codes and registration invitation codes.
  - Registered user list query, balance modification, permission changes, and self-deletion protection mechanism.

---

## 🛠️ Tech Stack

- **Frontend**: Vue 3 + Vite + Vue Router + Pinia + Vue-i18n
- **Backend**: Node.js + Express + JWT Authentication
- **Database**: SQLite3 (Using `better-sqlite3` for high-performance synchronous queries and transaction support)

## 🚀 One-Click Production Deployment

The project includes an out-of-the-box deployment script that automatically detects the environment, installs dependencies, compiles the frontend Vue code, and starts the backend Node service via the PM2 daemon (since the backend is configured for static hosting, no extra Nginx deployment is needed).

**System Requirements**: Linux / macOS server with Node.js (`v16+`) installed.

### 1. Clone and Enter the Project Directory
```bash
git clone https://github.com/ninjaz0/codeshop.git
cd codeshop
```

### 2. Run the One-Click Deployment Script
```bash
chmod +x deploy.sh
./deploy.sh
```

> **Done!** After successful deployment, the service will run by default at `http://127.0.0.1:3000`. You can access your server's IP and this port directly in your browser.

## 📄 Common Operations Commands (PM2)

The deployment script automatically installs global `pm2` to daemonize the backend process:

- **Check service status**: `pm2 status`
- **View real-time console logs**: `pm2 logs virtual-mall`
- **Restart service**: `pm2 restart virtual-mall`
- **Stop service**: `pm2 stop virtual-mall`
