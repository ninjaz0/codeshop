# 🚀 虚拟商品商城 (Virtual Goods Store / 发卡平台)

> **注：本项目全程由 AI 协助开发与构造。**

这是一个轻量级、现代化的虚拟商品自动发卡与兑换平台。系统采用前后端分离架构，专注于虚拟物品（如软件激活码、会员兑换券、教程文本等）的自动发货与交易流程。用户无需通过真实法币支付，而是通过核销系统后台生成的“充值兑换码”来增加账户余额，并使用余额在商城内购买数字商品。

---

## ✨ 核心特性

- **🌍 完美的多语言支持 (i18n)**
  - 系统内置并完全适配 5 种语言：简体中文、繁體中文、English、日本語、Русский。
  - 用户可随时在首页自由切换语言。

- **💳 闭环的虚拟经济体系**
  - **充值系统**：管理员在后台生成带有特定面值的“兑换码”。
  - **余额消费**：用户输入兑换码获取账户余额，并使用余额在商城内完成购物闭环。

- **📦 智能自动发发货系统**
  - 支持 **“唯一卡密 (Unique Key)”** 模式：一机一码，每售出一份自动扣减库存并向用户下发独立卡密。
  - 支持 **“固定内容 (Fixed Content)”** 模式：无限库存，所有购买该商品的用户将获得相同的下载链接或固定文本。

- **🔐 严谨的邀请与安全机制**
  - **邀请码注册**：新用户注册必须使用管理员后台生成的“邀请码”，有效控制滥用。
  - **事务级数据保护**：购买扣款、库存扣减、兑换码核销全程由 SQLite 数据库事务 (Transaction) 保护，严防并发超售和数据异常。

- **🛠️ 强大的后台管理大盘 (Admin Dashboard)**
  - 商品上架/下架管理、卡密库存导入。
  - 充值兑换码、注册邀请码批量生成与一键复制。
  - 注册用户列表查询、余额修改、权限变更与防自杀机制保护。

---

## 🛠️ 技术栈

- **前端 (Frontend)**: Vue 3 + Vite + Vue Router + Pinia + Vue-i18n
- **后端 (Backend)**: Node.js + Express + JWT 鉴权
- **数据库 (Database)**: SQLite3 (使用 `better-sqlite3` 提供高性能的同步查询与事务支持)

## 🚀 生产环境一键部署

项目内包含了一个开箱即用的一键部署脚本，它会自动检测环境、安装依赖、编译前端 Vue 代码并通过 PM2 守护进程启动后端 Node 服务（由于后端配置了静态托管，无需额外部署 Nginx）。

**系统要求**：Linux / macOS 服务器，且已安装 Node.js (`v16+`)。

### 1. 下载并进入项目目录
```bash
git clone https://github.com/ninjaz0/codeshop.git
cd codeshop
```

### 2. 运行一键部署脚本
```bash
chmod +x deploy.sh
./deploy.sh
```

> **完成！** 部署成功后，服务默认运行在 `http://127.0.0.1:3000`。你可以直接在浏览器中访问你的服务器 IP 和该端口。

## 📄 常用运维命令 (基于 PM2)

部署脚本会自动安装全局的 `pm2` 来守护后端进程：

- **查看服务运行状态**: `pm2 status`
- **查看实时控制台日志**: `pm2 logs virtual-mall`
- **重启服务**: `pm2 restart virtual-mall`
- **停止服务**: `pm2 stop virtual-mall`
