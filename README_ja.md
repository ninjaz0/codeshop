# 🚀 バーチャルモール (Virtual Goods Store / Codeshop)

> **注：このプロジェクトはAIの支援により完全に構築・開発されました。**

これは軽量でモダンな仮想商品の自動配信および引き換えプラットフォームです。フロントエンドとバックエンドの分離アーキテクチャを採用し、仮想アイテム（ソフトウェアのライセンスキー、会員バウチャー、チュートリアルテキストなど）の自動配信と取引プロセスに焦点を当てています。ユーザーは実際の法定通貨で支払う必要はなく、管理者がバックエンドで生成した「引き換えコード」を使用してアカウント残高を増やし、その残高でストア内のデジタル商品を購入します。

---

## ✨ 主な機能

- **🌍 完全な多言語サポート (i18n)**
  - システムは5つの言語（簡体字中国語、繁体字中国語、英語、日本語、ロシア語）に完全に適応しています。
  - ユーザーはホームページでいつでも自由に言語を切り替えることができます。

- **💳 クローズドループの仮想経済システム**
  - **チャージシステム**: 管理者がバックエンドで特定の額面を持つ「引き換えコード」を生成します。
  - **残高の消費**: ユーザーは引き換えコードを入力してアカウント残高を取得し、ストアでのショッピングに使用します。

- **📦 スマート自動配信システム**
  - **「個別キー (Unique Key)」モード**: 1つにつき1つの固有キー。販売されるごとに自動的に在庫を減らし、ユーザーに個別のキーを発行します。
  - **「固定コンテンツ (Fixed Content)」モード**: 在庫は無限。この商品を購入したすべてのユーザーは、同じダウンロードリンクまたは固定テキストを受け取ります。

- **🔐 厳格な招待およびセキュリティメカニズム**
  - **招待コード登録**: 新規ユーザーの登録には、管理者が生成した「招待コード」が必須であり、不正利用を効果的に防ぎます。
  - **トランザクションレベルのデータ保護**: 購入時の残高控除、在庫の控除、引き換えコードの使用は、すべて SQLite データベースのトランザクションによって保護され、同時販売による在庫超過やデータ異常を厳格に防ぎます。

- **🛠️ 強力な管理ダッシュボード (Admin Dashboard)**
  - 商品の公開/非公開管理、および仮想キー在庫のインポート。
  - チャージ引き換えコードと登録招待コードの一括生成およびワンクリックコピー。
  - 登録ユーザーリストの照会、残高変更、権限変更、および誤削除防止メカニズム。

---

## 🛠️ テクノロジースタック

- **フロントエンド**: Vue 3 + Vite + Vue Router + Pinia + Vue-i18n
- **バックエンド**: Node.js + Express + JWT 認証
- **データベース**: SQLite3 (`better-sqlite3` を使用して高性能な同期クエリとトランザクションをサポート)

## 🚀 本番環境へのワンクリックデプロイ

プロジェクトには、環境の自動検出、依存関係のインストール、Vue フロントエンドコードのコンパイル、および PM2 デーモンを介した Node バックエンドの起動を行う「ワンクリックデプロイメントスクリプト」が含まれています（バックエンドは静的ファイルをホストするように設定されているため、Nginx を別途展開する必要はありません）。

**システム要件**: Node.js (`v16+`) がインストールされている Linux / macOS サーバー。

### 1. プロジェクトをクローンしてディレクトリに移動
```bash
git clone https://github.com/ninjaz0/codeshop.git
cd codeshop
```

### 2. デプロイスクリプトを実行
```bash
chmod +x deploy.sh
./deploy.sh
```

> **完了！** デプロイが成功すると、サービスはデフォルトで `http://127.0.0.1:3000` で実行されます。ブラウザでサーバーの IP とこのポートに直接アクセスできます。

## 👑 初期管理者アカウント

システムが初めて起動し、データベースが初期化されると、デフォルトの最高権限管理者アカウントが自動的に作成されます。

- **ユーザー名**: `admin`
- **パスワード**: `admin123`

*(⚠️ 初回ログイン後、右上の「設定」からデフォルトのパスワードを変更することを強くお勧めします。)*

## 📄 よく使う運用コマンド (PM2)

デプロイスクリプトは、バックエンドプロセスをデーモン化するためにグローバルに `pm2` を自動インストールします：

- **サービスクテータスの確認**: `pm2 status`
- **リアルタイムコンソールログの表示**: `pm2 logs virtual-mall`
- **サービスの再起動**: `pm2 restart virtual-mall`
- **サービスの停止**: `pm2 stop virtual-mall`
