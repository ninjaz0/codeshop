# 🚀 虛擬商品商城 (Virtual Goods Store / 發卡平台)

> **注：本專案全程由 AI 協助開發與構造。**

這是一個輕量級、現代化的虛擬商品自動發卡與兌換平台。系統採用前後端分離架構，專注於虛擬物品（如軟體啟動碼、會員兌換券、教學文字等）的自動發貨與交易流程。使用者無需透過真實法幣支付，而是透過核銷系統後台生成的「儲值兌換碼」來增加帳戶餘額，並使用餘額在商城內購買數位商品。

---

## ✨ 核心特性

- **🌍 完美的多語言支援 (i18n)**
  - 系統內建並完全適配 5 種語言：簡體中文、繁體中文、English、日本語、Русский。
  - 使用者可隨時在首頁自由切換語言。

- **💳 閉環的虛擬經濟體系**
  - **儲值系統**：管理員在後台生成帶有特定面額的「兌換碼」。
  - **餘額消費**：使用者輸入兌換碼獲取帳戶餘額，並使用餘額在商城內完成購物閉環。

- **📦 智能自動發貨系統**
  - 支援 **「唯一卡密 (Unique Key)」** 模式：一機一碼，每售出一份自動扣減庫存並向使用者下發獨立卡密。
  - 支援 **「固定內容 (Fixed Content)」** 模式：無限庫存，所有購買該商品的使用者將獲得相同的下載連結或固定文字。

- **🔐 嚴謹的邀請與安全機制**
  - **邀請碼註冊**：新使用者註冊必須使用管理員後台生成的「邀請碼」，有效控制濫用。
  - **事務級資料保護**：購買扣款、庫存扣減、兌換碼核銷全程由 SQLite 資料庫事務 (Transaction) 保護，嚴防並發超售和資料異常。

- **🛠️ 強大的後台管理大盤 (Admin Dashboard)**
  - 商品上架/下架管理、卡密庫存匯入。
  - 儲值兌換碼、註冊邀請碼批量生成與一鍵複製。
  - 註冊使用者列表查詢、餘額修改、權限變更與防自殺機制保護。

---

## 🛠️ 技術棧

- **前端 (Frontend)**: Vue 3 + Vite + Vue Router + Pinia + Vue-i18n
- **後端 (Backend)**: Node.js + Express + JWT 鑑權
- **資料庫 (Database)**: SQLite3 (使用 `better-sqlite3` 提供高效能的同步查詢與事務支援)

## 🚀 生產環境一鍵部署

專案內包含了一個開箱即用的一鍵部署腳本，它會自動檢測環境、安裝依賴、編譯前端 Vue 程式碼並透過 PM2 守護進程啟動後端 Node 服務（由於後端配置了靜態託管，無需額外部署 Nginx）。

**系統要求**：Linux / macOS 伺服器，且已安裝 Node.js (`v16+`)。

### 1. 下載並進入專案目錄
```bash
git clone https://github.com/ninjaz0/codeshop.git
cd codeshop
```

### 2. 執行一鍵部署腳本
```bash
chmod +x deploy.sh
./deploy.sh
```

> **完成！** 部署成功後，服務預設運行在 `http://127.0.0.1:3000`。你可以直接在瀏覽器中訪問你的伺服器 IP 和該連接埠。

## 📄 常用運維命令 (基於 PM2)

部署腳本會自動安裝全域的 `pm2` 來守護後端進程：

- **查看服務運行狀態**: `pm2 status`
- **查看即時控制台日誌**: `pm2 logs virtual-mall`
- **重啟服務**: `pm2 restart virtual-mall`
- **停止服務**: `pm2 stop virtual-mall`
